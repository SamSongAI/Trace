Why Trace

一条痕迹，比一条待办更有价值

待办只告诉你“要做什么”，痕迹还保留了上下文、来源和下一步动作。Trace 的目标不是替代你的知识库，而是确保每个念头在消失前都留下可复用的线索。

Leave

任何应用里一键唤起，不打断当前任务，先把想法留下来。

Find

纯 Markdown 输出让每条记录都能被全文搜索、链接、引用和版本管理。

Route

Daily、Inbox、编辑器、Agent 共享同一份痕迹，不需要重复搬运。

System

从瞬时想法，到可执行轨迹

Trace 只负责最薄的一层输入，把思考稳定写入文件系统。后续整理、分发、自动化交给你现有的 Markdown 工具链。

1

Capture fast

快捷键唤起，Note / Clip / Link / Task / Project 五个模块保持捕捉结构最小但足够有用。

2

Persist clearly

默认写入 Daily，也支持直接新建 Inbox 文档。每条记录带时间和结构，方便回溯。

3

Route later

Obsidian、VS Code、Cursor、CloudCode 或自定义 Agent 都能直接消费这些 Markdown 痕迹。
