安装步骤

macOS

1) 通过 GitHub Releases 下载官方 `Trace.dmg`

2) 打开 DMG 后拖拽 `Trace.app` 到 Applications

3) 首次启动后在 Settings 里指定笔记库路径

Windows（即将开放）

1) 下载官方安装包

2) 完成安装并授予必要权限

3) 配置路径后即可开始留下 Trace
