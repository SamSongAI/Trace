隐私政策（Draft）

生效日期：2026-02-28

1. 本地优先

Trace 默认将内容写入你指定的本地 Markdown 路径，不依赖远端数据库。

2. 数据采集

当前版本不采集个人内容数据，不上传笔记正文。

3. 下载与更新

官网提供 GitHub Release 链接与校验信息。你可选择是否更新到新版本。

4. 联系方式

如有隐私相关问题，请通过官方产品渠道联系。
